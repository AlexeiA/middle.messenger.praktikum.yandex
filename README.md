## Описание

Учебный проект.

## Инструкция

- `npm install` - установка (не)стабильной версии,
- `npm run test` - тестирование,
- `npm run build` - сборка,
- `npm run start` - запуск.

Docker-релиз доступен по адресу https://alexeia-messenger.herokuapp.com/.
